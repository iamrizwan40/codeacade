HyperText Markup Language

Hypertext Markup Language (HTML):

HTML is the most basic thing you need to learn before you are gonna diving deep into web development. It just helps us to include the text, images, inputs, videos, graphics, etc that are displayed in the browser. It describes and defines the content of a webpage along with the basic layout of the webpage. When you have learned HTML you are gonna use other web technologies such as CSS, javaScript, etc. CSS allows us to change the odd layout of HTML into dynamically and interactvely looking web page, it helps to apply presentation over the html elements whereas Javascript a programming language that allows us to create dynamics and interactivites on the HTML elements.

HyperText is a key word in HTML to know about. It refers to the links that are connected to other web pages. HTML includes a bunch of links that lets the user go to desired page.

HTML uses a bunch of elements such as <head>, <title>, <body>, <header>, <footer>, <article>, <section>,<p> , <span>, <img>, <aside>, <audio>, <canvas>, <nav>, <video>,<div>, <a>, etc.
